DataIO.jl

A Julia port of the DataIO R package.

Introduction

This package provides functions for reading and writing of data files consisting of [n,k] matrices.

Installation

DataIO is not yet registered. To install the development version from a Julia REPL type ] to enter Pkg REPL mode and run
